# Neon Arkanoid

Classic brick-breaker arcade game with a modern neon aesthetic.

## How to Play

- **Mouse** — move paddle left/right
- **Click** or **Space** — start game / launch ball
- **Arrow Keys / A, D** — move paddle (layout-independent)

## Features

- 9 unique level patterns (levels 1-7 randomized)
- Neon glow effects and particle explosions
- Screen shake on brick destruction
- Synth sound effects
- Ball trail and smooth animations
- Ball speed increases with each level

## Run

Open `arkanoid.html` in any modern browser.

```bash
open arkanoid.html
```
